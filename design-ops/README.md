# Design Operation

Streamline how design work moves through your organization. Structured critique, clean handoffs, sprint planning, and quality gates that produce shipped work instead of endless discussions.

## Skills

| Skill | Description |
|-------|-------------|
| `ops-design-critique` | Conduzir críticas de design estruturadas, com frameworks de feedback claros e resultados acionáveis. |
| `ops-design-handoff` | Criar especificações de handoff com medidas, comportamentos, assets e casos de borda. |
| `ops-sprint-plan` | Planejar e facilitar design sprints, do enquadramento do desafio até o teste de protótipos. |
| `ops-design-review` | Estabelecer etapas de revisão com critérios, checklists e fluxos de aprovação. |
| `ops-QA` | Criar checklists de QA para verificar fidelidade da implementação ao design. |

## Commands

| Command | Description |
|---------|-------------|
| `/run-critique` | Run a structured critique with scoring and prioritized action plan. |
| `/ship-handoff` | Prepare a complete handoff package — specs, assets, and QA checklist. |
| `/plan-sprint` | Plan a design sprint from challenge definition to Day 5 test logistics. |
| `/setup-review` | Set up a structured design review process with criteria and approval flows. |
